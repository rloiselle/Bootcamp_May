About Me.

I am an analytical thinker with a natural curiosity and a passion for science and technology. I believe that transformation is something that happens when people connect and have the opportunity to learn and to be inspired.

I am interested in empowering individuals and small businesses by building the tools which serve to educate and inspire innovation.

My current and growing areas of expertise are: Web development technologies, Spatial and tabular data analysis and visualization, Environmental forecast modeling, Education, and Community outreach.

Projects

Awesome Hoodie Project

E-commerce website to raise money for local non-profit organizations. The site was migrated from bootstrap into a Sinatra appliaction. It is fully integrated with the PayPal payment system and data from PayPal's instant payment notification is stored into a database where the program administrator and merchant can coordinate on order fufillment.

The Puzzler

This is a simple javascript puzzle built with click handlers. It has a surprise feature when you solve the puzzle!

Roller Derby Stat App

This ongoing independent project: A rails application utilizing Ruby, R, and d3.js to summarize and display stats about roller derby teams and skaters from the Rinxter.net API.

Bird Song Memory

Memorize bird songs with this interactive puzzle! This is another ongoing independent project which uses JavaScript to dynamically embed bird songs into a tiled game board from the Xeno-canto API. The goal of the game is to find the audio match and score points.
